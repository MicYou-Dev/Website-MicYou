# Markdown Extension Examples

This page demonstrates some of the markdown extensions provided by VitePress.

## Syntax Highlighting

VitePress provides syntax highlighting for code blocks:

```js
export default {
  name: 'MicYou',
  features: ['low-latency', 'cross-platform', 'high-quality']
}
```

## Custom Containers

::: info
This is an info box.
:::

::: tip
This is a tip.
:::

::: warning
This is a warning.
:::

::: danger
This is a dangerous warning.
:::
