# Markdown 擴展示範

此頁面展示了 VitePress 提供的一些 Markdown 擴展功能。

## 語法高亮

VitePress 為程式碼區塊提供語法高亮：

```js
export default {
  name: 'MicYou',
  features: ['low-latency', 'cross-platform', 'high-quality']
}
```

## 自訂容器

::: info
這是一個資訊框。
:::

::: tip
這是一個提示。
:::

::: warning
這是一個警告。
:::

::: danger
這是一個危險警告。
:::
